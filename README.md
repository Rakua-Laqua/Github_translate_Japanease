# GitHub 日本語化（辞書置換）

`github.com` の **UIラベル**（タブ名/ボタン/メニュー/補助ラベル等）を、**英→日**の辞書置換で日本語化する Chrome/Edge 拡張（MV3）です。

外部翻訳APIは使いません（外部通信なし）。

## できること

- 辞書（完全一致）で UI テキストを日本語化
- SPA/動的DOM差し替えに追従して翻訳を適用
- 辞書にない「UIっぽい英語」を未知語ログとして収集（ローカル保存）
- Popup で ON/OFF・ログ収集 ON/OFF・ログコピー/消去
- Options で除外セレクタ、属性翻訳、ユーザー辞書編集、ログのエクスポート

## 翻訳しないもの（事故防止）

- ユーザー生成コンテンツ（README/Issue/PR本文、コメント、コード差分など）
- `pre`, `code`, `.markdown-body` 配下（初期設定の除外）
- `input` / `textarea` の内部テキスト（入力系は原則スキップ）

## インストール（開発用の読み込み）

1. Chrome/Edge で `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」を ON
3. 「パッケージ化されていない拡張機能を読み込む」
4. このリポジトリの `Github_translate_Japanease` フォルダを選択

## 使い方

- GitHub を開くと自動で翻訳が走ります
- 右上の拡張アイコン → Popup から
	- 日本語化 ON/OFF
	- 未知語ログ収集 ON/OFF
	- 未知語ログのコピー（JSON/CSV）
	- 未知語ログの全消去

詳細設定は Options（拡張機能の詳細 → 拡張機能のオプション）から行えます。

## 設定項目（Options）

- 除外セレクタ: 初期値 `pre, code, .markdown-body`
- 翻訳対象属性: `aria-label` / `title` / `placeholder`
- ログ収集: 最大件数、収集文字数範囲
- ユーザー辞書: JSON（英語キー → 日本語値）で上書き/追記
- 未知語ログ: 簡易表示、JSON/CSVエクスポート、辞書テンプレ生成

## データ保存

- `chrome.storage.sync`
	- 有効/無効、除外セレクタ、ログ収集ON/OFF、属性ON/OFF、ユーザー辞書など
- `chrome.storage.local`
	- 未知語ログ（ローカルのみ）

## プライバシー

- 外部通信は行いません（翻訳API未使用）
- 収集ログはローカル保存のみ（エクスポートはユーザー操作時のみ）
